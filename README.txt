Short description
-----------------

PCPIN Template is a web template renderer engine for PHP-based software. 

Benefits of using PCPIN Template engine: 
 - Separates PHP code from HTML
 - Faster application development
 - Better source code organisation
 - Easier teamwork (PHP developers and HTML designers)



Requirements
------------

Server:
- PHP 4.3 or later

-----
